## Known Limitations

- This gate optimizes the Python DQ fallback used when native host DQ is not
  preferred. An optimized native or GPU-resident DQ map builder remains a future
  target.
- The fast path assumes GLASS resident CUDA generated finite integer-valued
  count maps. Direct helper calls and non-resident arbitrary arrays keep strict
  NaN/fractional handling.
- This gate does not change calibration formulas, registration, warp,
  winsorized rejection, DQ bit definitions, integration math, accepted-frame
  decisions, or output science pixels.
