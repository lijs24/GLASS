# GLASS v0.1.0 Windows Alpha 1

This is the first Windows-oriented GLASS alpha release.

## Included

- Portable Windows package: unzip and run `glass.cmd`.
- `glass doctor` runtime diagnostics.
- FITS/XISF metadata scan and processing-plan generation.
- Synthetic FITS dataset generation.
- CPU baseline calibration, registration helpers, local normalization, and integration paths.
- CUDA-aware Python package structure and runtime capability detection.
- Wheel and source distribution for Python users.

## Windows Requirements

- Windows x64.
- Python is bundled in the portable package.
- NVIDIA driver is only required for CUDA-enabled local builds or future GPU binary packages.
- CUDA Toolkit, Visual Studio Build Tools, CMake, and Ninja are not required for the portable CPU-capable package.

## Quick Start

```powershell
.\glass-doctor.cmd
.\glass.cmd --help
.\glass.cmd synthetic --out runs\demo_synth --frames 8 --width 128 --height 128 --filter H --known-shift
.\glass.cmd audit --root runs\demo_synth --out runs\demo_audit --backend cpu
```

## Notes

- This alpha package is intended to validate installation, CLI ergonomics, metadata handling, reporting, and CPU-baseline workflows on ordinary Windows machines.
- Public GPU binary packaging is planned as a follow-up once the CUDA compatibility matrix is settled.
